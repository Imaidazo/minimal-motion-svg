# Minimal Motion SVG Documentation

Welcome to the official documentation for **Minimal Motion SVG**.

Minimal Motion SVG is an open-source collection of lightweight, modern and accessible animated SVG components for web and mobile applications.

The library uses pure SVG and embedded CSS animations, with transparent backgrounds and no external runtime dependencies.
